# 🎯 Interactive Online Quiz Maker

A dynamic, front-end web application that allows users to create custom quizzes and test their knowledge interactively. Built entirely with Vanilla web technologies without relying on heavy frameworks.

**🔴 Live Demo:** [Play the Quiz Maker Here](https://ranjeetpatel9598-wq.github.io/online-quiz-maker/)

---

## ✨ Features
* **Custom UI Design:** Unique, engaging user interface with a custom-coded progress bar and modern card layouts.
* **Quiz Creation:** Users can input their own questions, multiple-choice options, and define correct answers.
* **Interactive Gameplay:** Step-by-step question rendering with real-time selection feedback.
* **Result Analysis:** Detailed score tracking and a review section to check correct/incorrect answers at the end.
* **Local Storage Database:** Completely serverless! Uses browser `localStorage` to save user credentials and quiz data locally.

## 🛠️ Tech Stack
* **HTML5:** Semantic structure and form handling.
* **CSS3:** Custom styling, flexbox layouts, hover effects, and responsive design.
* **JavaScript (ES6):** DOM manipulation, state management, array mapping, and local storage integration.
